Guide Dog Snark Generator

This tool generates a sharp response for common guide dog etiquette failures, then backs it up with an explanation that tells people why this is not cool.

It is part small web tool, part educational device, and part public service announcement with teeth.

Build a response

Choose a scenario and a snark level, then activate Generate response. If you miss a required choice, the form announces an error message automatically.

What this tool is doing

  • It pairs a guide dog etiquette scenario with a response that matches the selected snark level.
  • It does not preselect the scenario or snark level, so the user stays in control of the input instead of inheriting a default choice.
  • It provides an announced error message if required selections are missing, helping people understand what needs fixing without hunting through the form.
  • It follows the snark with a plain-language explanation so the tool stays educational as well as sharp-tongued.
  • It uses native form controls, visible focus states, keyboard-friendly interaction, and predictable structure that support WCAG-aligned use.
  • It uses ARIA live regions for dynamic updates, so generated responses, copy status messages, and validation feedback are announced when they change.
