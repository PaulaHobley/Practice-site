Guide Dog Snark Generator

This tool generates a sharp response for common guide dog etiquette failures, then backs it up with an explanation that tells people why this is not cool.

It is part small web tool, part educational device, and part public service announcement with teeth.

What this page is showcasing

  • It combines structured content design, UI writing, and front-end logic in a single small tool with a clear purpose.
  • It uses custom CSS illustration and state-based animation to create three distinct dog reactions without relying on image files.
  • It turns a simple form input into dynamic output, pairing scenario data, randomised responses, and explanatory text in real time.
  • It keeps the interaction user-controlled by avoiding default selections and by validating only when a response is requested.
  • It demonstrates accessible form design through native controls, clear labels, visible focus states, and announced validation feedback.
  • It uses stable ARIA labelling and live regions so the dog reaction, generated response, copy status, and error handling are announced predictably.
  • It shows tidy component thinking by separating data, presentation, animation states, and interaction logic into readable sections of code.

Experimental Web Playground is a practice space for building accessible, hand-coded web tools with personality.
