Welcome to my experimental web playground

This site exists because I wanted somewhere to build things properly, break them safely, and figure out what actually works without someone else’s template getting in the way.

Why this site exists

I’m learning to code in a way that makes sense to me, which means understanding how things work under the hood and making sure everything works without relying on visual assumptions.

What you’ll find here

  • Small web tools and utilities
  • Layout and styling experiments
  • Accessibility testing

How I’m approaching this

I’m focusing on reliability, accessibility, and simplicity rather than complexity for the sake of it.
