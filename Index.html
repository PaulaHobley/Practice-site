Welcome to my experimental web playground

This site exists because I wanted somewhere to build things properly, break them safely, and figure out what actually works without someone else’s template getting in the way. It’s part learning exercise, part testing ground, and part “what happens if I just build the thing myself and see how it behaves.”

Why this site exists

I’m learning to code in a way that makes sense to me, which means understanding how things work under the hood, building layouts that don’t fall apart the moment you zoom, making sure interaction still works when you’re not relying on a mouse, and avoiding unnecessary complexity just because it looks impressive. This site is where I put that into practice.

What you’ll find here

At the moment, it’s a mix of experiments and ideas that may or may not behave themselves.

  • Small web tools and utilities
  • Layout and styling experiments
  • Accessibility-focused testing
  • Random projects that seemed like a good idea at the time

Some things will be polished. Some things will be held together with metaphorical duct tape. That’s the point.

How I’m approaching this

I’m not trying to build the fanciest site on the internet. I’m trying to build one that works reliably, behaves properly at different zoom levels, doesn’t rely on visual assumptions, uses native HTML where it makes sense, and only adds complexity when it actually solves a problem.

If something feels over-engineered, it probably is.

Accessibility note

Accessibility isn’t an afterthought here. It’s part of the foundation.

That means proper structure, predictable navigation, usable interaction patterns, and making sure things still make sense when you’re not relying on visuals. No gimmicks. No pretending a plugin fixed everything.

Ongoing work

This site will evolve as I learn more.

Things will change, improve, and occasionally break before they get better. If something doesn’t work yet, it’s probably on the list.

Closing

If you’re here, you’re either curious, lost, or I sent you the link.

Either way, welcome.

This is a work in progress, which is a polite way of saying I’m building as I go.
